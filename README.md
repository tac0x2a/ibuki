# ibuki

![Ibuki](./frontend/src/assets/ibuki_logo.svg)

Ibuki is a scraping platform.

# Frontend
```
$ npm install -g @vue/cli@next
$ cd frontend
$ npm install
$ vue upgrade --next
$ npm run dev

# access http://localhost:3000
```

# Backend
```
$ npm install -g @nestjs/cli
$ cd backend
$ npm install
$ npm run start
# access http://localhost:3001
```
